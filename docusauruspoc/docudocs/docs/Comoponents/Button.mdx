# Button Props

| Prop | Type | Default | Required | Description |
|-----|:------|:--------|:--------|:------------|
| label | string | `'Click me'` | Yes | just some text The label of the button. |
| <i>nestedProp</i> | NestedProp | - | No | A nested property with 4 levels of depth. |
| <i>onClick</i> | object | `() => {}` | No | The click handler for the button. |
| type | One of ["danger", "primary", "secondary", "success"] | `'secondary'` | Yes | The type of the button. |
